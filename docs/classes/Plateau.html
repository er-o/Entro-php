getScore()

getScore(\Joueur  $joueur) : array

Fonction donnant le score d'un joueur Le score est défini de la façon suivant : Nombre de pions bloqués : c'est le nombre de pions impossible à bouger, mais qui ne sont pas isolé Nombre de pions isolés : C'est le nombre de pions impossible à bouger, mais qui n'ont pas d'autre pions autours d'eux Le joueur gagne si son nombre de pions bloqués vaut 7, et que sont nombre de pions isolés vaut 0

Parameters

\Joueur $joueur

Le joueur auquel on veut le joueur

Returns

array —

Une liste de deux cases contenant le score. Le nombre de pions isolés est dans la première case, et le nombre de pions bloqués dans la deuxieme

None found
